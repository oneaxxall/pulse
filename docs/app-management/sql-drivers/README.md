# 🛢 SQL Drivers

SQL app drivers are an abstraction that connects directly to relational database servers and retrieves app details from a specific database and table.

The documentation for each of the supported database systems includes instructions for how to create the proper database tables needed by Socket to manage apps.

{% hint style="info" %}
Behind the scenes, Socket uses [Knex](https://knexjs.org) to connect to relational databases and retrieve app data.
{% endhint %}
