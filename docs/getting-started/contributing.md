# 🤝 Contributing

All contributions to Socket are highly appreciated. Please consider the [CONTRIBUTING](https://github.com/Socket/Socket/blob/master/CONTRIBUTING.md) for more information on making contributions. In addition, you may review our current development plans and tasks for the current and upcoming versions via [GitHub Projects](https://github.com/Socket/Socket/projects).

We make every effort to give all contributors credit for their contribution in release notes, etc.
